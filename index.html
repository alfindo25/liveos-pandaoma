Panduan LiveOS

Selamat datang di LiveOS 👋
LiveOS adalah sistem pelacak performa live streaming Pandaoma secara real-time. Semua data tersimpan di cloud dan bisa diakses kapan saja.
➕
Input Data
Input manual atau import Excel

Dashboard
Lihat GMV, grafik, dan ranking
🤖
AI Chat
Analisis otomatis dari data

Incentive
Lihat incentive bulanan kamu
💡 Klik salah satu kartu di atas atau tab di bagian atas untuk mulai belajar fitur tertentu.
1
Pilih Marketplace
▶
Pilih salah satu dari tiga opsi:
🛍 Shopee untuk sesi live di Shopee dashboard
📦 DLD untuk penjualan Shopee yang tidak terbaca di dashboard (Di Luar Dashboard)
🎵 TikTok untuk sesi live di TikTok Shop
💡 DLD dihitung masuk ke GMV Shopee di dashboard.
2
Pilih Nama Streamer
▶
Ketik nama streamer di kolom pencarian. Nama harus dipilih dari daftar — tidak bisa diketik bebas.

Kalau nama streamer belum ada, tambahkan dulu di menu Data Streamer (hanya admin).
⚠️ Nama streamer yang tidak ada di master data akan ditolak sistem.
3
Isi Tanggal, Jam, Durasi
▶
Tanggal: tanggal sesi live berlangsung (otomatis hari ini).
Jam Mulai: jam mulai live (format HH:MM, contoh: 19:00).
Durasi: total durasi live dalam menit (contoh: 120 = 2 jam).
4
Isi GMV dan Catatan
▶
GMV: total nilai transaksi sesi live (dalam Rupiah, tanpa titik/koma). Boleh diisi 0 kalau tidak ada penjualan.

Catatan: kondisi yang terjadi selama live — misal "flash sale", "koneksi putus", "giveaway berhasil", dll.
💡 Catatan sangat penting! AI Chat bisa merangkum dan menganalisis catatan untuk evaluasi performa.
5
Import via Excel (untuk data banyak)
▶
1. Klik ⬇ Download Template .xlsx
2. Buka file di Excel / Google Sheets
3. Isi data sesuai format kolom (jangan ubah header)
4. Simpan file, lalu upload di bagian Import
5. Cek preview → klik Import Data
⚠️ Nama streamer di Excel harus sama persis dengan yang ada di master data.
1
Filter Marketplace
▶
Di bagian atas ada 3 tombol filter marketplace: Shopee DLD TikTok
Klik untuk toggle aktif/nonaktif. Klik lagi untuk mematikan. Bisa dikombinasikan — misal lihat Shopee+DLD saja.
💡 DLD (Di Luar Dashboard) adalah penjualan Shopee yang tidak terbaca sistem mereka — tetap dihitung di GMV Shopee.
2
Filter Periode
▶
Pilih range waktu: Hari Ini, Kemarin, 7 Hari Terakhir, 30 Hari Terakhir, atau Kustom (pilih tanggal sendiri).
Semua metric card, grafik, dan tabel rank mengikuti filter yang dipilih.
3
Metric Cards
▶
4 kartu utama:
Total GMV: total semua marketplace di periode yang dipilih
GMV Shopee: termasuk DLD
GMV TikTok: khusus TikTok Shop
Total Sesi: jumlah sesi live sesuai filter
4
Grafik GMV Harian
▶
Grafik garis menampilkan tren GMV per hari. Merah = Shopee+DLD, Biru = TikTok.
Grafik mengikuti filter periode yang dipilih. Hover ke titik untuk lihat nilai detail.
5
Rank Streamer & Rank Sesi
▶
Rank Streamer Bulan Ini: ranking berdasarkan total GMV bulan berjalan.
Rank Sesi Live (Top 5): 5 sesi dengan GMV tertinggi sesuai filter aktif.
Klik tombol Detail → untuk melihat semua sesi.
1
Filter & Urutkan Data
▶
Filter marketplace (Shopee/DLD/TikTok) dan periode (Hari Ini/Kemarin/7 Hari/30 Hari/Kustom).
Klik header kolom untuk sort: Streamer, MP, GMV, Tanggal, Jam, Durasi.
2
Kolom Catatan
▶
Catatan yang panjang ditampilkan singkat. Klik tombol detail untuk baca catatan lengkap dalam popup.
💡 Isi catatan sedetail mungkin — AI Chat bisa merangkumnya untuk evaluasi.
3
Edit & Hapus Sesi
▶
Streamer hanya bisa edit/hapus data yang mereka sendiri input.
Admin (superuser) bisa edit/hapus semua data.
4
Export Excel
▶
Klik ⬇ Export Excel untuk download semua data yang tampil (sesuai filter aktif) ke file .xlsx.
File berisi kolom: Streamer, Marketplace, GMV, Tanggal, Jam, Durasi, Catatan, Diinput oleh.
1
Apa itu menu Incentive?
▶
Menu Incentive menampilkan data incentive bulanan yang diterima setiap streamer.
Data ini bersifat rahasia — setiap streamer hanya bisa melihat data milik sendiri.
Admin bisa melihat dan menginput semua data.
2
Cara melihat Incentive (Streamer)
▶
1. Login dengan email yang sudah didaftarkan ke admin
2. Buka menu Incentive
3. Data incentive kamu otomatis muncul
4. Gunakan filter tahun (◀ 2026 ▶) dan filter bulan untuk melihat periode tertentu
5. Klik beberapa bulan sekaligus untuk melihat total gabungan
💡 Akunmu harus dikaitkan dulu oleh admin ke nama streamer di Data Streamer.
3
Cara input Incentive (Admin)
▶
1. Buka menu Incentive
2. Klik tombol + Input Incentive di pojok kanan atas
3. Pilih streamer, bulan, tahun
4. Isi nominal dan catatan (opsional)
5. Klik 💾 Simpan
⚠️ Pastikan email streamer sudah diisi di Data Streamer agar mereka bisa melihat data incentive-nya.
1
Apa yang bisa AI Chat lakukan?
▶
AI Chat bisa menganalisis data sesi live kamu secara otomatis, termasuk:
• Tren GMV per streamer atau marketplace
• Rangkuman catatan sesi dalam rentang tanggal tertentu
• Poin bagus dan improvement berdasarkan catatan
• Rekomendasi strategi ke depan
• Perbandingan performa antar streamer
2
Contoh pertanyaan ke AI
▶
💬 "Siapa streamer terbaik bulan Mei?"
💬 "Apa saja poin bagus & improvement dari catatan tgl 1-15 Mei 2026?"
💬 "Rangkum catatan live Shopee bulan ini"
💬 "Jam berapa performa live paling tinggi?"
💬 "Bandingkan GMV Shopee vs TikTok minggu ini"
💡 Semakin detail catatan yang diisi saat input, semakin akurat analisis AI.
3
Setup API Key Groq
▶
AI Chat membutuhkan Groq API Key untuk berfungsi.
1. Buka Setting
2. Scroll ke bagian Gemini API Key
3. Paste API key kamu → klik Simpan
API key gratis bisa didapatkan di console.groq.com
1
Tambah Streamer Baru
▶
Hanya admin yang bisa tambah streamer.
1. Isi Nama Lengkap (sesuai nama yang akan dipakai saat input data)
2. Isi Role (contoh: Host Utama, Host Pendukung)
3. Isi Email Login — ini digunakan untuk akses menu Incentive
4. Klik Tambah Streamer
💡 Email streamer harus sama dengan email yang digunakan untuk login ke LiveOS.
2
Edit Data Streamer
▶
Klik tombol Edit di sebelah nama streamer.
Bisa ubah nama, role, dan email tanpa menghapus data sesi yang sudah ada.
⚠️ Jangan hapus streamer yang sudah punya data sesi — data sesi tidak akan hilang tapi nama streamer akan kosong.
1
Groq API Key
▶
Paste API key dari console.groq.com untuk mengaktifkan AI Chat.
Key disimpan di browser lokal — tidak dikirim ke server.
2
Approve User Baru (Admin)
▶
Setiap akun baru yang daftar butuh persetujuan admin sebelum bisa masuk.
Di Setting → bagian Approve User Baru → klik Approve atau Tolak.
3
Logout
▶
Scroll ke bawah di Setting → klik Logout.

Standar Operasional Host Live

Berlaku untuk semua sesi live di Shopee & TikTok

5 Standar Host yang Unggul
1

Berpenampilan rapi & menarik

Wajib pakai makeup. Penampilan mencerminkan profesionalisme brand Pandaoma di depan penonton.

Wajib setiap sesi
2

Senyum & Ekspresif

Ekspresi wajah yang hangat dan antusias membuat penonton betah dan nyaman di live kamu.

Energi positif
3

Suara & artikulasi jelas

Bicara dengan tempo stabil, tidak terburu-buru. Penonton harus bisa menangkap setiap kata dengan mudah.

Komunikasi
4

Live dengan Semangat

Energi host menular ke penonton. Kalau host antusias, penonton juga ikut terbawa dan lebih aktif.

Energi
5

Menguasai Produk

Pahami fungsi, keunggulan, cara pakai, dan masalah umum tiap produk sebelum live. Penonton percaya kalau kamu benar-benar paham.

Product knowledge
Panduan Teknik Live
6

Mata selalu menatap layar

Tatap kamera terus — ini menciptakan koneksi visual dengan penonton. Jangan sering melirik ke bawah atau ke samping.

Koneksi penonton
7

Bangun komunikasi 2 arah

Jangan hanya presentasi satu arah. Tanya kebutuhan dan masalah penonton secara aktif agar mereka merasa diperhatikan.

Engagement
8

Sebutkan benefit checkout

Selalu ingatkan penonton apa yang mereka dapatkan saat checkout — bonus, promo, gratis ongkir, voucher, dll.

Konversi
9

Angkat produk saat demo

Saat memperlihatkan produk, angkat ke depan kamera dengan posisi yang jelas. Jangan diletakkan begitu saja di meja.

Presentasi produk
10

Duduk tegak, jangan lesu

Postur tubuh yang baik menunjukkan kepercayaan diri dan menambah kesan profesional di mata penonton.

Postur
Closing & transisi etalase
11

Closing sebelum pindah etalase

Sebelum berpindah ke produk berikutnya, selalu tutup dengan closing yang mendorong penonton untuk segera checkout.

Contoh script: "Yang mau produk ini, ada di etalase 1 ya — checkout sekarang mumpung ada promo spesial banget. Oke, kita lanjut ke etalase 5!"
